## Recommended Content Structure

### 5-Minute Morning Mobility Guide

**Page 1: Title & Introduction**
- Title: "5-Minute Morning Mobility Guide"
- Subtitle: "Wake Up Pain-Free with Dr. Maks Birikov, DPT"
- Brief intro about why morning mobility matters

**Pages 2-6: Exercises (one per page)**
1. Cat-Cow Stretch (1 minute)
2. Hip 90/90 Stretch (1 minute)
3. Thoracic Rotation (1 minute)
4. World's Greatest Stretch (1 minute)
5. Shoulder CARs (1 minute)

Each page should include:
- Exercise name
- Clear illustration or photo
- Step-by-step instructions
- Common mistakes to avoid

**Page 7: Call to Action**
- Brief message about booking a session
- Contact information
- Website URL
- QR code to booking page
